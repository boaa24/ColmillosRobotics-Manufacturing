Cómo usar esto

  1. Sube tus archivos (PDF/DXF/STEP + .nc/.gcode) al repo, por ejemplo en /files/.
  2. Regístralos en data/files.json para que aparezcan en “Archivos”.
  3. Edita data/bom.json para tu BOM real (ensambles → piezas → cantidad).
  4. Edita data/tasks.json para tu plan (Gantt).
  5. Habilita GitHub Pages (Settings → Pages) y listo.

Limitación importante

Esto es un sitio estático: no hay base de datos. Por eso el avance se guarda local por navegador. Para compartir avance: Export/Import.
